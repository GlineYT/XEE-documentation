VARIABLES

XMLEE has a few variable types. These are listed below
Name Children Description
int ✗ Signed 64-bit integer.
float ✗ Double-precision floating-point number.
as per IEEE 754 specification.
bool ✗ A 1-bit boolean value.
str ✗ An arbitrary length string.
Multi-byte characters should be represented correctly.
Immutable.
arr ✓ An array of any type of variable.
All children are evaluated and added in order of declaration.
dict ✓ A mapping of keys to values.
In implementation, every variable type should be able to be both a key and a value.
Expects pair for children while defining.
set ✓ A unique, unordered set of elements.
All children are evaluated and added.
func ✓ A function, with args and an element to evaluate when called.
These can be either defined by the user or imported from a built-in library.
Evaluate the inner block with call.
The arguments are added to scope respective to their order in args.
null ✗ A null value.
break ✗ Breaking constant. Return this inside a loop to break out.
continue ✗ Continuing constant. Return this inside of a loop to jump to the next iteration.

Ints and floats

Integers are whole numbers and work similarly to integers in any other programming language
As with any other variable, they are defined using the assign keyword
They are defined as such:
5
Note: Integer literals in XEE are decimal only.
Binary (0b), octal (0o), and hexadecimal (0x) prefixes are not supported.
Use the radix function to parse binary/hex strings if needed.
The value in between the opening and closing tag is the value fo the integer
Floats are defined in the same manner

Strings

Str represents a string of characters of an arbitrary length
Supports multi-byte characters, e.g. UTF-16
Additionally, a range of string manipulation functions are availible
more information in the standart library documentation
hello
Strings can be concatenated with other variable types
This is similar to python's "f-string". It is done using a array
See the below code for reference
Your answer is 10 and that is False

While the join function allows for a array with a arbitrary amount of children

Arrays

Arrays represent a array of any type of variable.
1 2
Arrays can be concatenated using the "concat" function in the standart library
To nest one array into another, use the add tag.
<import id="!std" /> <import id="!io" /> 1 2 3 4
The above code outputs [1,2,3,4]
The below code, however, would output [1,2,[3,4]]
1 2 3 4
This is useful when working with hierarchical or nested data structures
For example parsing json,XML,HTML or other such hierarchical files
