UNARY OPERATORS

Unary operators are tags that take exactly one child and perform an operation on it.
They are built-in language tags that operate directly on a single value.
Unary operators in XEE provide:
  • Negation (neg) — flips the sign of a number
  • Absolute value (abs) — returns the magnitude of a number
  • Bitwise NOT (not) — inverts bits of an integer or boolean
Note: Unary operators are tags, not function calls. They take exactly one child element.

Operator Reference

Tag Operation Input Types Result Description
<neg> -x Integer, Float Same type as input Negates the value (flips the sign).
5 → -5
-3.14 → 3.14
<abs> |x| Integer, Float Same type as input Returns the absolute value (magnitude).
-5 → 5
3.14 → 3.14
<not> ~x or !x Integer, Boolean Integer or Boolean Bitwise NOT for integers.
Logical NOT for booleans.
5 → -6 (bitwise)
true → false

Examples

Negation (neg)

<assign id="positive"> <neg> <int>-5</int> </neg> </assign> <!-- Result: 5 --> <assign id="negative"> <neg> <float>3.14</float> </neg> </assign> <!-- Result: -3.14 -->

Absolute Value (abs)

<assign id="distance"> <abs> <int>-42</int> </abs> </assign> <!-- Result: 42 --> <assign id="magnitude"> <abs> <float>-3.14159</float> </abs> </assign> <!-- Result: 3.14159 -->

Bitwise/Logical NOT (not)

<!-- Bitwise NOT on integer --> <assign id="inverted_bits"> <not> <int>5</int> </not> </assign> <!-- Result: -6 (two's complement) --> <!-- Logical NOT on boolean --> <assign id="is_not_admin"> <not> <bool>true</bool> </not> </assign> <!-- Result: false -->

Combined Example

<!-- Get absolute value of a negative number --> <assign id="safe_value"> <abs> <neg> <int>100</int> </neg> </abs> </assign> <!-- Result: 100 --> <!-- Check if something is NOT true --> <if> <not> <access id="is_admin"/> </not> <call id="println"><str>You are not an admin</str></call> <call id="println"><str>Welcome, admin!</str></call> </if>
Note: The not operator performs a bitwise NOT on integers
and a logical NOT on booleans. For integers, ~5 = -6 (all bits inverted).
To negate the sign, use the neg tag
Warning: All unary operators require exactly one child.
Providing more than one child will result in a runtime error.
