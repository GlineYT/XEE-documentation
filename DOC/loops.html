LOOPS

XEE provides two types of loops: <for> and <while>.
Both are essential for iteration and repetition in your programs. Loops in XEE offer:
  • for — iterates over arrays, strings, ranges, and dictionaries
  • while — repeats a block while a condition is true
  • break and continue — control flow inside loops

FOR LOOP

The <for> tag iterates through an indexable object.
It takes two children:
the indexable object
a block of code to execute for each iteration.
Tag Children Description
<for> 2 Iterates through an indexable object.
If given an integer, iterates from 0 to that number (exclusive).
A variable with the set ID is created for each iteration.
The variable persists after the loop ends.

Syntax

<for id="item"> <access id="iterable"/> <!-- First child: the indexable object --> <block> <!-- Second child: the loop body --> <call id="println"><access id="item"/></call> </block> </for>

Examples

Iterating over an Array

<assign id="fruits"> <arr> <str>apple</str> <str>banana</str> <str>cherry</str> </arr> </assign> <for id="fruit"> <access id="fruits"/> <block> <call id="println"><access id="fruit"/></call> </block> </for>

Iterating over a Range (Integer)

<for id="i"> <int>5</int> <!-- 0, 1, 2, 3, 4 --> <block> <call id="println"> <call id="join"> <arr> <str>Iteration: </str> <access id="i"/> </arr> </call> </call> </block> </for>

Iterating over a Dictionary (Keys)

<assign id="user"> <dict> <pair><str>name</str><str>Gameline</str></pair> <pair><str>age</str><int>25</int></pair> </dict> </assign> <for id="key"> <access id="user"/> <block> <call id="println"> <call id="join"> <arr> <access id="key"/> <str>: </str> <access id="user"> <access id="key"/> </access> </arr> </call> </call> </block> </for>

Nested For Loops

<for id="i"> <int>3</int> <block> <for id="j"> <int>3</int> <block> <assign id="coord"> <call id="join"> <arr> <str>(</str> <access id="i"/> <str>, </str> <access id="j"/> <str>)</str> </arr> </call> </assign> <call id="println"><access id="coord"/></call> </block> </for> </block> </for>
Note: The loop variable persists after the loop ends.
It holds the last value from the iteration.

WHILE LOOP

The <while> tag keeps looping the second child (the body)
until the first child (the condition) returns false.
Tag Children Description
<while> 2 Keeps looping the second child until the first child returns false.
The condition is evaluated before each iteration.

Syntax

<while> <condition/> <!-- First child: evaluated each iteration --> <block> <!-- Second child: executed while condition is true --> <!-- loop body --> </block> </while>

Examples

Basic Counter

<assign id="count"><int>0</int></assign> <while> <lt> <access id="count"/> <int>5</int> </lt> <block> <call id="println"><access id="count"/></call> <assign id="count"> <add> <access id="count"/> <int>1</int> </add> </assign> </block> </while>

Countdown

<assign id="countdown"><int>5</int></assign> <while> <gt> <access id="countdown"/> <int>0</int> </gt> <block> <call id="println"><access id="countdown"/></call> <assign id="countdown"> <sub> <access id="countdown"/> <int>1</int> </sub> </assign> </block> </while>

Array Iteration with While

<assign id="fruits"> <arr> <str>apple</str> <str>banana</str> <str>cherry</str> </arr> </assign> <assign id="idx"><int>0</int></assign> <assign id="len"> <call id="len"><access id="fruits"/></call> </assign> <while> <lt> <access id="idx"/> <access id="len"/> </lt> <block> <assign id="fruit"> <access id="fruits"> <access id="idx"/> </access> </assign> <call id="println"><access id="fruit"/></call> <assign id="idx"> <add> <access id="idx"/> <int>1</int> </add> </assign> </block> </while>

BREAK AND CONTINUE

XEE supports <break/> and <continue/> tags for controlling loop execution.
Tag Description
<break/> Exits the current loop immediately.
<continue/> Skips the rest of the current iteration and jumps to the next.

Example

<!-- Print numbers 0 to 9, skip 5, stop at 8 --> <for id="i"> <int>10</int> <block> <if> <eq><access id="i"/><int>5</int></eq> <continue/> </if> <if> <eq><access id="i"/><int>8</int></eq> <break/> </if> <call id="println"><access id="i"/></call> </block> </for> <!-- Output: 0 1 2 3 4 6 7 -->
Note: The <break/> and <continue/> tags are self-closing and have no children.
Warning: When incrementing a counter in a <while> loop, always use <add>
(not join) to keep values as integers.
