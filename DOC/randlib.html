RANDOM LIBRARY

The Random library provides functions for generating random numbers,strings and arrays
. To use it, import !rand in your program.
<import id="!rand"/>

Function Reference

Name Arguments Returns Description
rand None Float Returns a random floating-point number between 0 and 1.
guid None String Generates a random UUID (version 4).
rand_str length, base String Generates a random string of specified length
Uses characters from the given base (2-36).
shuffle array Array Returns a new array with the elements randomly shuffled.
choice array Any Returns a random element from the array.
sample count Array Returns an array of random floats with the specified count.

rand

Returns a random floating-point number between 0.0 and 1.0.
<assign id="random_num"> <call id="rand"/> </assign> <call id="println"> <add> <str>Random number: </str> <access id="random_num"/> </add> </call>

guid

Generates a random UUID (version 4) string. Useful for unique identifiers.
<assign id="user_id"> <call id="guid"/> </assign> <call id="println"> <add> <str>User ID: </str> <access id="user_id"/> </add> </call>

rand_str

Generates a random string of the specified length using characters from the given base.
Base ranges:
  • Base 2 → binary (0-1)
  • Base 10 → decimal (0-9)
  • Base 16 → hexadecimal (0-9, a-f)
  • Base 36 → alphanumeric (0-9, a-z)
<!-- Binary string (8 bits) --> <assign id="binary"> <call id="rand_str"> <int>8</int> <int>2</int> </call> </assign> <!-- Hexadecimal string (16 chars) --> <assign id="hex"> <call id="rand_str"> <int>16</int> <int>16</int> </call> </assign> <!-- Alphanumeric string (12 chars) --> <assign id="alnum"> <call id="rand_str"> <int>12</int> <int>36</int> </call> </assign> <call id="println"> <add> <add> <add> <str>Binary: </str> <access id="binary"/> </add> <str>\nHex: </str> </add> <access id="hex"/> </add> </call>

shuffle

Returns a new array with the elements randomly shuffled.
The original array is unchanged.
<assign id="original"> <arr> <str>apple</str> <str>banana</str> <str>cherry</str> <str>date</str> </arr> </assign> <assign id="shuffled"> <call id="shuffle"> <access id="original"/> </call> </assign> <call id="println"> <add> <str>Shuffled: </str> <access id="shuffled"/> </add> </call>

choice

Returns a random element from the array.
<assign id="fruits"> <arr> <str>apple</str> <str>banana</str> <str>cherry</str> </arr> </assign> <assign id="random_fruit"> <call id="choice"> <access id="fruits"/> </call> </assign> <call id="println"> <add> <str>Random fruit: </str> <access id="random_fruit"/> </add> </call>

sample

Returns an array containing the specified number of random floats between 0 and 1.
<assign id="samples"> <call id="sample"> <int>5</int> </call> </assign> <call id="println"> <add> <str>5 random floats: </str> <access id="samples"/> </add> </call>
Note: The random number generator is cryptographically secure where available.
For cryptographic purposes, consider using the Crypto library instead.
