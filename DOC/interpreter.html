THE INTERPRETER

Since XEE is a interpeted language, it uses a interpeter. The interpeter is written in rust
and is a fork of the original XMLang language and interpeter originally created by Balt.
The source code is availible on the Github repository. As of now, there is only a Linux version availible.
To use it, follow these steps:
  1. Download the interpeter
  2. Copy it to a folder, preferably the same one as the program you will run
  3. In a terminal, type ./xmlang-ee-linux ./program.xee
