MATH LIBRARY

The Math library provides:
  • mathematical constants
  • trigonometric functions
  • logarithms
  • other mathematical operations.

    • To use it, import !math in your program.
    <import id="!math"/>

    Constants

    Name Value Description
    pi 3.141592653589793 Archimedes's constant.
    The ratio of a circle's circumference to its diameter.
    e 2.718281828459045 Euler's number. Base of natural logarithms.
    inf ∞ Positive infinity. Larger than any number.
    nan NaN Not a Number.
    Result of undefined operations (e.g., 0/0).
    <assign id="circle_area"> <mul> <access id="pi"/> <mul> <access id="radius"/> <access id="radius"/> </mul> </mul> </assign>

    Unary Functions

    Function Description
    sin, cos, tan Trigonometric functions (radians)
    asin, acos, atan Inverse trigonometric functions
    sinh, cosh, tanh Hyperbolic functions
    asinh, acosh, atanh Inverse hyperbolic functions
    exp Exponential (e^x)
    ln Natural logarithm (base e)
    log2, log10 Base-2 and base-10 logarithms
    sqrt Square root
    ceil, floor, round, trunc Rounding functions
    fract Fractional part of a number
    sign Sign of a number (-1, 0, 1)
    deg, rad Convert between degrees and radians
    <assign id="angle_rad"> <call id="rad"> <int>90</int> </call> </assign> <assign id="sin_val"> <call id="sin"> <access id="angle_rad"/> </call> </assign> <call id="println"> <add> <str>sin(90°) = </str> <access id="sin_val"/> </add> </call>

    Binary Functions

    Function Arguments Description
    atan2 y, x Four-quadrant inverse tangent
    hypot x, y Euclidean distance (sqrt(x² + y²))
    max a, b Returns the larger of two numbers
    min a, b Returns the smaller of two numbers
    log a, b Logarithm of a with base b
    <assign id="distance"> <call id="hypot"> <int>3</int> <int>4</int> </call> </assign> <!-- distance = 5 --> <assign id="larger"> <call id="max"> <int>10</int> <int>20</int> </call> </assign> <!-- larger = 20 -->

    Ternary Function

    Function Arguments Description
    lerp a, b, t Linear interpolation between a and b at factor t (0-1)
    <assign id="halfway"> <call id="lerp"> <int>0</int> <int>100</int> <float>0.5</float> </call> </assign> <!-- halfway = 50 -->

    Examples

    Calculate Circle Circumference

    <assign id="radius"><int>5</int></assign> <assign id="circumference"> <mul> <int>2</int> <mul> <access id="pi"/> <access id="radius"/> </mul> </mul> </assign> <call id="println"> <add> <str>Circumference: </str> <access id="circumference"/> </add> </call>
    Note: All trigonometric functions work in radians.
    Use rad to convert degrees to radians, and deg to convert radians to degrees.
    Integer arguments are automatically converted to floating-point numbers.
