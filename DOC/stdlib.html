STANDARD LIBRARY

The Standard Library provides core functionality including:
  • I/O
  • string manipulation
  • array operations
  • dictionary manipulation
  • type conversion
  • utility functions
<import id="!std"/>

I/O Functions

Name Arguments Returns Description
print value None Prints a value to stdout without a newline.
println value None Prints a value to stdout with a newline and flushes.
flush None None Flushes stdout buffer.
input None String Reads a line from stdin, strips whitespace.
sleep seconds None Pauses execution for the specified number of seconds.
time None Float Returns the current Unix timestamp as a float.

String & Array Manipulation

Name Arguments Returns Description
join array String Converts array elements to strings and concatenates them.
split string, delimiter Array Splits a string by delimiter into an array.
split_at array, index Array of two arrays Splits an array into two halves at the specified index.
concat array1, array2 Array Concatenates two arrays element-wise (not nested).
substr string, start, end String Extracts a substring from start (inclusive) to end (exclusive).
len value Integer Returns the length of a string, array, dict, or set.

Dictionary Functions

Name Arguments Returns Description
dict_merge dict1, dict2 Dictionary Merges two dictionaries (second overwrites first on key conflicts).
dict_set dict, key, value Dictionary Returns a new dictionary with the key-value pair added/updated.
dict_get dict, key Any or null Returns value for key, or null if not found.
dict_remove dict, key Dictionary Returns a new dictionary with the key removed.
dict_has dict, key Boolean Returns true if the key exists in the dictionary.
dict_keys dict Array Returns an array of all keys in the dictionary.
dict_values dict Array Returns an array of all values in the dictionary.
dict_items dict Array Returns an array of [key, value] pairs.

Type Conversion & Utility

Name Arguments Returns Description
chr code String Converts a Unicode code point to a single-character string.
ord character Integer Converts a single-character string to its Unicode code point.
radix value, base String or Integer Converts numbers between bases 2-36 (string↔integer).
type value String Returns the type name of the value.
hash value Integer Returns a non-cryptographic hash (for hash tables, not security).
range start, end, step Array Creates an array of integers from start to end (exclusive) by step.
