CRYPTO LIBRARY

The Crypto library provides:
  • cryptographic hashing
  • HMAC
  • password hashing functions.
To use it, import !crypto in your program.
<import id="!crypto"/>

Functions

Name Arguments Returns Description
crypto_hash algorithm, input String (hex) Computes a hash of the input string.
Uses the algorithm specified
crypto_hmac algorithm, key, input String (hex) Computes HMAC-SHA256.
bcrypt_hash password, cost String (bcrypt hash) Hashes a password using bcrypt
takes a cost factor (4-31).
bcrypt_verify password, hash Boolean Verifies a password against a bcrypt hash.

Supported Algorithms

Algorithm Output length Use case
md5 32 hex chars Legacy / checksums (NOT for security)
sha256 64 hex chars Security / integrity
sha512 128 hex chars Security / integrity

crypto_hash

Computes a cryptographic hash of the input string.
<assign id="md5_hash"> <call id="crypto_hash"> <str>md5</str> <str>Hello World</str> </call> </assign> <assign id="sha256_hash"> <call id="crypto_hash"> <str>sha256</str> <str>Hello World</str> </call> </assign> <call id="println"> <add> <str>MD5: </str> <access id="md5_hash"/> </add> </call>

crypto_hmac

Computes HMAC-SHA256 for message authentication.
Requires a secret key.
<assign id="hmac_result"> <call id="crypto_hmac"> <str>sha256</str> <str>my_secret_key</str> <str>message to authenticate</str> </call> </assign>

bcrypt_hash

Hashes a password using bcrypt.
<assign id="password_hash"> <call id="bcrypt_hash"> <str>my_secure_password</str> <int>12</int> </call> </assign> <call id="println"> <add> <str>Hash: </str> <access id="password_hash"/> </add> </call>

bcrypt_verify

Verifies a password against a stored bcrypt hash.
Returns true if the password matches.
<assign id="is_valid"> <call id="bcrypt_verify"> <str>my_secure_password</str> <access id="password_hash"/> </call> </assign> <call id="println"> <add> <str>Password valid: </str> <if> <access id="is_valid"/> <str>true</str> <str>false</str> </if> </add> </call>
Note: bcrypt hashes are automatically salted.
The cost factor determines the number of iterations (2^cost).
A cost of 12 means 4096 iterations.
Security Warning: For password storage, always use bcrypt
