CONDITIONALS

XEE provides a single conditional construct: the <if> tag. It is a 3-in-1 construct that combines:
if
else if
else
switch functionality.
 The <if> tag works as follows:
  • Children are paired as condition and expression
  • For each pair, the condition is evaluated first
  • If the condition is true, the corresponding expression is executed and the <if> returns
  • If no condition is true and there is an odd number of children the last child is executed as an else
Key Point: There is no separate else or else if tag.
The structure of the <if> tag itself provides this functionality.

Syntax

If (2 children)

<if> <condition/> <!-- If true --> <expression/> <!-- Execute this --> </if>

If-Else (3 children)

<if> <condition/> <!-- If true --> <expression1/> <!-- Execute this --> <expression2/> <!-- Else (odd child) --> </if>

If-Else If-Else (4+ children)

<if> <condition1/> <!-- If true --> <expression1/> <!-- Execute this --> <condition2/> <!-- Else if true --> <expression2/> <!-- Execute this --> <condition3/> <!-- Else if true --> <expression3/> <!-- Execute this --> <expression4 (else) /> <!-- Else (odd child) --> </if>

Examples

Basic If (2 children)

<assign id="value"><int>10</int></assign> <if> <gt> <access id="value"/> <int>5</int> </gt> <call id="println"><str>Value is greater than 5</str></call> </if>

If-Else (3 children)

<assign id="value"><int>3</int></assign> <if> <gt> <access id="value"/> <int>5</int> </gt> <call id="println"><str>Value is greater than 5</str></call> <call id="println"><str>Value is 5 or less</str></call> </if>

If-Else If-Else (4+ children)

<assign id="score"><int>85</int></assign> <if> <gt> <access id="score"/> <int>90</int> </gt> <call id="println"><str>Grade: A</str></call> <gt> <access id="score"/> <int>80</int> </gt> <call id="println"><str>Grade: B</str></call> <gt> <access id="score"/> <int>70</int> </gt> <call id="println"><str>Grade: C</str></call> <call id="println"><str>Grade: F</str></call> </if>

Switch-Case Pattern (Multiple Equality Checks)

<assign id="command"><str>start</str></assign> <if> <eq> <access id="command"/> <str>start</str> </eq> <call id="println"><str>Starting...</str></call> <eq> <access id="command"/> <str>stop</str> </eq> <call id="println"><str>Stopping...</str></call> <eq> <access id="command"/> <str>status</str> </eq> <call id="println"><str>Status: Running</str></call> <call id="println"><str>Unknown command</str></call> </if>
Important: The <if> tag evaluates condition-expression pairs in order
and returns the first matching expression.
Do NOT use multiple separate <if> tags as they will cause the function to return early.
Use one <if> with multiple condition-expression pairs.
